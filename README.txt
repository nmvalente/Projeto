Para arrancar a aplicação de Backup Service basta correr um script com as configurações aí existentes. 
Cada script faz arrancar um peer, que é simples de identificar até pelo próprio nome do ficheiro script. 

Se se pretender alterar os endereços e as portas, basta abrir esse ficheiro e fazer as respetivas alterações.
Se se pretender correr um único script também é possível bastando para isso configurar até um limite de 3 peers por máquina.
